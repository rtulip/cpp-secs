Detailed Description

EntityBuilder is a builder class which constructs an entity with components.

In order to more easily allow for entities to have a set of components this builder class is used. Components can easily be added by using .with<T>(T &&t), which comsumes t. Upon calling .build() the Entity is added to the World which it's a part of.
